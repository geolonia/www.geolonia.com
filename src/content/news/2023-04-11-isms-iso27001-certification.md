株式会社Geoloniaは、情報セキュリティマネジメントシステム（ISMS）の国際規格認証「ISO27001」を取得したことをお知らせいたします。

ISMS（Information Security Management System）認証とは、組織が自社で保護すべき情報資産に対して、国際規格「ISO/IEC 27001」に基づいて適切に管理および運用されていることを認証する制度です。

Geoloniaでは、今回のISMS認証取得により、今後もISO認証規格に基づいた適切な情報管理や運用および改善を継続し、お客様が安心して利用できるセキュアなサービス構築に努めて参ります。

## 認証の概要

| 会社名 | 株式会社Geolonia |
| --- | --- |
| 認証基準 | JIS Q 27001：2014 （ISO/IEC 27001:2013） |
| 認証番号 | GIJP-0969-IC |
| 認証範囲 | ウェブ地図のデザイン・開発、位置情報開発支援、住所ソリューション、空間IDの開発 |
| 認定機関 | 株式会社GCERTI-JAPAN |
| 認証登録日 | 2023年2月21日 |
| 情報セキュリティ方針 | https://geolonia.com/information-security/ |
