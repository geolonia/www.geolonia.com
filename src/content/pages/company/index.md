## 株式会社Geoloniaについて

株式会社Geoloniaは、位置情報テクノロジーとオープンソースを通じて社会の課題を解決することをミッションとしています。

### 会社情報

| 項目 | 内容 |
|------|------|
| 会社名 | 株式会社Geolonia |
| 設立 | 2019年 |
| 代表者 | 代表取締役 |
| 所在地 | 東京都 |
| 事業内容 | 地図・位置情報サービスの提供<br>地理空間データ基盤の開発<br>オープンソースソフトウェアの開発 |

### ビジョン

位置情報テクノロジーを活用し、より便利で持続可能な社会の実現に貢献します。

### ミッション

オープンソースの力で、地理空間データをすべての人が活用できるようにします。
